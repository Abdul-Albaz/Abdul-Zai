Image storage on GitHub

Cover images you pick, and your whole collection (games/music/books), are committed straight into your GitHub repo — images into an imges folder, your entries into data/library.json — so it's all part of the live site, not just this browser. Open the site from any other PC and it will load the same data from GitHub automatically. This needs a token with write access to that one repo.

Create one at github.com/settings/personal-access-tokens/new → pick this repository → under Permissions set Contents: Read and write → Generate, then paste it above. The token is saved only in this browser's storage; it is never written into the site's files.
